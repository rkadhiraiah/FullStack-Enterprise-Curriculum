## 🛠️ Teaching Methodology
I believe in a **Project-First** approach. My cohorts do not just write code; they build deployable systems. Every module concludes with:
1. **Architectural Design:** Diagramming the solution before coding.
2. **Implementation:** Writing clean, testable code using JUnit/Mockito.
3. **Deployment:** Automating CI/CD pipelines via Jenkins and Docker.
