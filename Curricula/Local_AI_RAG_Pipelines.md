# Course Outline: Private AI Infrastructure & RAG Pipelines

**Target Audience:** Mid-level to Senior Developers
**Duration:** 2 Weeks (Hands-on Workshop)

## Course Objective
To equip developers with the skills to architect, deploy, and maintain secure, local AI solutions using open-source models, entirely independent of commercial cloud APIs.

## Module Breakdown

### Week 1: Foundational Local AI & The Ollama Framework
* **Day 1:** Introduction to Local LLMs vs. Cloud APIs (Security and Cost implications).
* **Day 2:** Hardware considerations (CPU vs. GPU, VRAM requirements) and system setup.
* **Day 3:** Installing and configuring the Ollama framework.
* **Day 4:** Pulling and running localized models (focusing on Qwen 2.5 and Llama 3).
* **Day 5:** Exposing local models via REST endpoints.

### Week 2: Building the Retrieval-Augmented Generation (RAG) Pipeline
* **Day 1:** Introduction to Vector Databases (ChromaDB / FAISS) and Embeddings.
* **Day 2:** Document ingestion, chunking strategies, and processing text data.
* **Day 3:** Creating the generation loop: Connecting the Vector DB to the Local LLM.
* **Day 4:** Python pipeline construction and API integration.
* **Day 5:** Capstone Project: Deploying a localized, secure HR workflow assistant.

## Assessment
* Code review of the capstone Python pipeline.
* Successful demonstration of querying private data without internet egress.
