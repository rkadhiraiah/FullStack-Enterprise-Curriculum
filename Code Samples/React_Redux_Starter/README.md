# ⚛️ React + Redux Toolkit (RTK) Starter Template

**Objective:** A clean, production-ready boilerplate used in Week 2 of the Enterprise MERN Bootcamp to teach global state management.

This sample demonstrates how to structure a modern React application using Redux Toolkit for efficient, boilerplate-free state management, adhering strictly to SOLID principles.

## 🛠️ Tech Stack
* React 18 (Functional Components & Hooks)
* Redux Toolkit (RTK)
* React-Redux

## 🚀 Key Learning Outcomes
1. **Slice Architecture:** Encapsulating state, reducers, and actions in a single file.
2. **Store Configuration:** Centralized store setup with automatic middleware integration.
3. **Immutability:** Leveraging Immer (built into RTK) for safe state mutations.
